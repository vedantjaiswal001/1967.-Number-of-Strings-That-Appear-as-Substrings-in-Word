# 1967.-Number-of-Strings-That-Appear-as-Substrings-in-Word
Count how many strings in patterns appear as substrings of word. Iterate through each pattern, use Python's in operator to check if it exists in word, and increment the count. Return the total. Time: O(n × m), where n is the number of patterns and m is the word length. Space: O(1).

by VEDANT JAISWAL
